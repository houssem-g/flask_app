************************************************Test the API thanks to Postman***********************************************************

1) Method GET

summary : this method is used just for read

	input : 
		- select the method GET 
		- paste this adress http://127.0.0.1:5000/readAllTable
	output 
		- If success you should receive all data stored in the database


2) Method Post

summary: We use this method to create new line in the Database

	input :
		- select the method Post
		- paste this adress http://127.0.0.1:5000/readAllTable
		- in the body add please an object to store in the DB
		- exemple object {"lat": 123, "lon": 456, "altitude": 999, "name": "Frank"}
	output :
		- If success you will receive a message "success=True"
		
3) Method PUT

summary: We use this method to update rows in the Database

	input :
		- select the method PUT
		- paste this adress http://127.0.0.1:5000/readAllTable
		- in the body add please an object to update the row in the DB
		- your object should contain AT LEAST the id of the row to update and at least one new value for any column to update
		- exemple object {"lat": 123, "lon": 456, "altitude": 999, "name": "Frank", "id": 4}
	output :
		- If success you will receive a message "success=True"

3) Method DELETE

summary: We use this method to delete row(s) from the Database

	input :
		- select the method DELETE
		- paste this adress http://127.0.0.1:5000/readAllTable
		- in the body add please an object which contain the id of the row to delete in the DB
		- exemple object {"id": 4}
	output :
		- If success you will receive a message "success=True"
